# work
###一、功能：
对灰度图像添加高斯 + 椒盐混合噪声，使用 BM3D 算法去噪，分别含噪和去噪图像，并计算 PSNR、SSIM
###二、参数说明
| 参数               | 说明                     |
|--------------------|--------------------------|
| DATASET_PATH       | 原始灰度图像路径         |
| NOISY_SAVE_PATH    | 含噪图像保存路径         |
| RESULT_SAVE_PATH   | 去噪结果保存路径         |
| GAUSSIAN_SIGMA     | 高斯噪声标准差（σ=25）|
| SALT_PEPPER_AMOUNT | 椒盐噪声比例（2%）|
###三、流程
1.读取灰度图像并归一化至 0-1 范围；
2.调用add_mixed_noise函数添加混合噪声同时保存含噪图像；
3.调用 BM3D 算法去噪，保存去噪图像；
4.计算 PSNR（峰值信噪比）、SSIM（结构相似性）;
###四、注意事项
1.仅处理灰度图像，图像均做 0-1 归一化处理；
2.输出图像命名规则：原文件名_mixed_noisy/denoised.png
